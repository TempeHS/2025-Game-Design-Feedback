## Game Object

The game object is the character each player controls in the fighting game. It has parts that let it move, jump, attack, and use special moves. It also keeps track of things like health and speed. The character reacts to player input and plays animations like walking or striking. Everything works together to make the character feel smooth and fun to play. It's built using Unity's tools like Rigibody2D for physics and Animator for cool movements.
